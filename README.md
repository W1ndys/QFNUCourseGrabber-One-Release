# QFNUCourseGrabberPy

> 曲阜师范大学 QFNU 抢课脚本 | 强智教务抢课脚本 | 强智教务 2017 | 大学抢课脚本 | 学院抢课脚本
>
> 本脚本以强智教务系统 2017 版本为基础，支持 2017 版本的所有功能，包括选修选课、专业内跨年级选课、本学期计划选课、公选课选课、计划外选课，其他版本未测试，但基本的请求 API 几乎类似，可以参考本脚本实现

## 如何使用

本程序已使用编译程序编译发布，无需配置环境

进入main文件夹，里面有一个main.exe文件，打开，输入你的学号，会生成一个base64请求码，把请求码发给我，我会给你一个文件，把这个文件放到和exe文件相同目录下，不要修改文件名
然后使用根目录下的 html 文件生成选课配置，生成后把config.json同样放到exe相同目录下
再次启动exe检查是否正常运行

可选：开启ocr_server.py会启动图像识别服务器，有Python基础的可以启动，实现自动识别验证码自动登录，若不开启，则需要手动输入验证码登录
